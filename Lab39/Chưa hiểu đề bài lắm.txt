Học viên hoàn thiện bài tập theo yêu cầu sau:

Xây dựng 2 project NodeJS với 2 Database khác nhau, đảm bảo các yêu cầu sau:

Project A: Xử lý chức năng đăng nhập, đăng xuất (Dùng passport.js hoặc bất kỳ cách nào)
Project B: Đồng bộ trạng thái đăng nhập của Project A
Giải thích:

Nếu Project A đăng nhập -> Project B cũng sẽ ở trạng thái đăng nhập
Nếu Project A chưa đăng nhập -> Project B cũng ở trạng thái chưa đăng nhập
Trong Project B không có table users
Trong Project B sẽ bổ sung thêm nút “Đăng nhập”, “Đăng xuất”. Khi User muốn đăng nhập, hoặc đăng xuất sẽ chuyển hướng qua Project A -> Sau khi hoàn thành bên Project A sẽ tự động chuyển hướng quay về Project B
